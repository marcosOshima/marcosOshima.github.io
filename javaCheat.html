JavaScript Cheatsheet

Property/Method Description
anchors Returns a collection of all elementsin the document that have a name attribute
assign() Loads a new document
break Exits a switch or a loop
ceil() Returns x, rounded up to the nearest integer
floor() Returns x, rounded down to the nearest integer
class Declares a class
error() Outputs an error message to the console
forEach() Call a function for each array element
getDate() Returns the day of the month(from 1 to 31)
getTime Returns the number of miliseconds since midnight January 1 1970 and a specified date
getElementByID Returns the element that has the ID attribute with the specified value
let Declares a variable inside brackets {} scope
onclick Event occurs when the user clicks on an element
onload Event occurs when object is loaded
parseInt() Parses a string and returns an integer
