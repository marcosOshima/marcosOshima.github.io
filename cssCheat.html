CSS Cheatsheet

Property Description
color Sets the color of text
background-color Sets the background color of an element
font-size Sets the size of the font
font-family Specifies the font family for text
margin Sets all the margin properties of an element
padding Sets all the padding properties of an element
height Sets the height of an element
width Sets the width of an element
opacity Sets the opacity level for an element
visibility Specifies if an element is visible or not
text-align Specifies the horizontal alignment of text
text-transform Controls the capitalization of text
word-break Specifies how words should break when reaching the end of a line
. Selects all elements in a determined class
# Selects all elements in a determined id
* Selects all elements
